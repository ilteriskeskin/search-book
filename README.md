# search-book
This console app is search book using Google Books

## Installation

```Bash
git clone https://github.com/ilteriskeskin/search-book.git
cd search-book/src
go build main.go
echo "alias booksearch='/home/ilteriskeskin/Belgeler/search-book/src/main'" >> ~/.bashrc # or ~/.zshrc
booksearch
```
